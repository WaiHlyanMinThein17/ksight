# Security Policy

## Supported versions

ksight is currently in early development. Security fixes are applied to
the latest release only.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

To report a security vulnerability, please do not open a public GitHub issue.
Instead, email waihlyanminthein@gmail.com with a description of the issue,
the steps to reproduce it, affected versions, and any known mitigations.

You can expect a response within 48 hours acknowledging the report.
