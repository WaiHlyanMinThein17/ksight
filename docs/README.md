# Demo assets

| File | Description |
|------|-------------|
| `demo.gif` | Animated demo shown in the project README |
| `screenshot.png` | Static dashboard screenshot |
| `demo.cast` | Original asciinema terminal recording |
| `demo-trimmed.cast` | Trimmed recording used to generate `demo.gif` |

The GIF was produced with [agg](https://github.com/asciinema/agg):
agg --theme monokai --font-size 16 --idle-time-limit 1 --speed 1.2 
demo-trimmed.cast demo.gif
